Webanalyze, utility to uncover technologies used on websites.
May 13, 2015
1 minute read

webanalyze is a (Go) port of wappalyzer, a browser extension to display technologies used on websites.

To use this in a performant way to test hundreds of hosts simultaneously I rewrote this project in Go. It’s not as accurate as the extension because of the missing javascript executing, but most of apps are found correctly.

webanalze can be used as a standalone tool (e.g. to generate a JSON output) or as a lib.

To keep app definitions up to date the list of apps definitions can be downloaded from the Wappalyzer repository with the -u (update) flag.

https://github.com/rverton/webanalyze
